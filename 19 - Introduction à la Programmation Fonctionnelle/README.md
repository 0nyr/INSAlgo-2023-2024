## Exercices

Faire les exercices suivants en écrivant seulement du code en style fonctionnel :
- pas de variables mutables
- pas de boucles
- uniquement des fonctions pures

Et en essayant de réutiliser des concepts vus dans les slides:
- composition
- pattern matching
- ADT
